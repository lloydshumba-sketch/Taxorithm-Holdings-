# Taxorithm Holdings Website Deployment Package

This folder is ready for static hosting.

## Files
- `index.html` — main website
- `assets/taxorithm-logo.svg` — local logo and favicon asset
- `netlify.toml` — Netlify deployment configuration
- `_headers` — recommended security/cache headers

## Netlify Deployment
1. Log in to Netlify.
2. Go to **Sites > Add new site > Deploy manually**.
3. Drag and drop this whole folder or the ZIP file.
4. Netlify will detect the form named `taxorithm-enquiry`.
5. Add a custom domain in **Domain management**.

## Notes
- The contact form is configured for Netlify Forms.
- Professional/academic affiliation logos are loaded remotely; replace them with official local assets if you obtain permission.
- Update the email/domain metadata once the final domain is confirmed.
