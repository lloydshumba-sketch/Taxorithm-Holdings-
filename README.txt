Taxorithm Advisory multipage website update

Upload these files to the root of the GitHub repository:

index.html
about.html
services.html
tax.html
registration.html
contact.html
style.css
script.js

Keep your existing policies.html file in the same root folder because all pages link to it.

Do not upload these files inside a subfolder. They must sit at the same level as policies.html and CNAME if CNAME exists.

GitHub Pages does not run PHP, so no PHP file is required. This website uses static HTML, CSS and JavaScript, which is correct for GitHub Pages.
