# Taxorithm Advisory multipage website

Upload these files to the root of your GitHub repository:

- index.html
- about.html
- services.html
- tax.html
- registration.html
- contact.html
- style.css
- script.js
- policies.html

Important: policies.html is not included in this package because you already have it. Upload your existing policies.html in the same root folder.

GitHub Pages does not run PHP. This site uses static HTML, CSS and JavaScript, which is correct for GitHub Pages.

After upload, your URLs will be:

- https://taxorithm.us/
- https://taxorithm.us/about.html
- https://taxorithm.us/services.html
- https://taxorithm.us/tax.html
- https://taxorithm.us/registration.html
- https://taxorithm.us/contact.html
- https://taxorithm.us/policies.html
